# movie-success-analysis
A data science project for predicting movie success and revenue using classification and regression models.



# Repository Structure

## Dataset

- `tmdb_preprocessed.csv`
  - Final preprocessed movie dataset used for regression and classification tasks.

## Code Documentation

- `regression_code_documentation.docx`
  - Documentation for the regression code and model implementation details.

- `classification_code_documentation.docx`
  - Documentation for the classification code and implementation details.

## Source Code

### Data Preprocessing

- `data_preprocessing.py`
  - Data preprocessing code including cleaning, feature engineering, encoding, scaling, and transformation.

- `data_preprocessing_no_print_ver.py`
  - A simplified preprocessing version without extensive print outputs.

### Regression

- `regression.py`
  - Regression pipeline code using Linear Regression, Random Forest, and XGBoost models.

### Classification

- `classification_pipeline.py`
  - Classification pipeline code using KNN and Decision Tree models.

## Additional Files

- `.gitignore`
  - Specifies files and folders that should not be tracked by Git.

- `README.md`
  - Overview and description of the project repository.
